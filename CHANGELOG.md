# 변경 이력

## [3.2.10] - 2026-03-03

### 변경
- 커스텀 일수 입력값이 즉시 저장되도록 개선 (`input/change/blur` 저장)
- 시간대 ID 생성/추가 시 충돌 방지 로직 추가
- 그룹 탭 선택 안정화 (`pointerup` 기반 선택, 스크롤 이동 임계치 적용, 키보드 선택 지원)
- 버전 및 리소스 캐시 쿼리 업데이트 (`v=334`)

## [3.2.9] - 2026-03-03

### 변경
- 시간 수정 패널 개선:
  - `-4주`, `+4주` 버튼 추가
  - 커스텀 일수 입력 UI를 `커스텀 일수 [-][숫자][+]` 형태로 개편
  - 커스텀 일수는 양수만 입력되도록 보정(최소 1일)
- 표기/복사 항목 버튼 내부 정렬 보정(드래그 아이콘, 체크박스, 텍스트, 항목 버튼 수직 정렬)
- 시간대 테이블 영역의 좌우 여백/배경 정렬 보정
- 텍스트 입력 박스 스타일을 커스텀 일수 입력 스타일과 동일 기준으로 통일
- 버전 및 리소스 캐시 쿼리 업데이트 (`v=333`)

## [3.2.7] - 2026-03-03

### 변경
- 표기/복사 양식 구조를 `시간대 / 이름 / UTC / 시간 / 일수 / 기간`으로 재구성
- `시간` 항목에 세부 체크박스 드롭다운 추가:
  - `낮/밤`, `날짜`, `시간`, `요일`
- 시간 컬럼 헤더를 항상 `시간`, `추가 시간`으로 통일
- 기본값 조정:
  - 표기 시간 세부 항목: 모두 활성화
  - 복사 시간 세부 항목: `날짜 + 시간`만 활성화
  - `DST 무시` 기본값 `true`
- 옵션 라벨(`시간 수정`, `DST 무시`, `추가 시간 표시`, `표기/복사 양식`)에 안내 툴팁 아이콘 추가
- 사이드바 하단 설정 영역 재배치:
  - 테마/언어
  - 설정 내보내기/가져오기
  - 모든 설정 초기화
- 그룹/도구/시간 테이블 간격 조정 및 UI 정렬 개선
- 표기/복사 항목 드래그 버튼 가시성/정렬 개선
- 버전 및 리소스 캐시 쿼리 업데이트 (`v=331`)
