# Changelog

이 문서는 Global Time Viewer의 주요 변경 사항을 기록합니다.

## [3.2.4] - 2026-03-03

### Added
- 그룹별 기준 시간 저장/복원 지원 (`group.baseTimezoneId`).
- 표준 시간대(IANA) 유효성 검증 추가.

### Changed
- 커스텀 시간대 오프셋 시간 콤보에서 음수 부호가 정상 표시되도록 수정 (`-05` 형식).
- 시간대 오프셋 계산 경로의 예외 안전성 보강.
- 버전 표기 및 리소스 캐시 쿼리 업데이트 (`v=326`).

## [3.2.3]

### Changed
- 버전 업데이트.

## [3.2.1]

### Changed
- 이미지 저장 파일명 규칙을 `그룹_시간대약어_날짜시간` 형식으로 변경.
- 이미지 저장 안정성 및 다운로드 처리 개선.
- 복사 실패 예외 처리 강화.
- 표기/복사 방식과 테이블 컬럼 표시 연동 개선.
- 라이트 모드 선택 UI 가독성 개선.

## [3.1.0]

### Changed
- 기준 시간대 기능 및 시간 수정 기능 확장.
- 기간(일수) 표시/복사 기능 확장.
- UI/복사 포맷 관련 다수 개선.
