# 🌐 Global Time Viewer v3.1.0

전 세계 여러 도시의 시간을 한눈에 비교하고 관리할 수 있는 프리미엄 세계 시간 뷰어입니다. 이제 **다국어 지원**과 **크롬 확장 프로그램** 기능을 통해 언제 어디서나 더 편리하게 세계 시간을 확인할 수 있습니다.

## ✨ 주요 기능

### 1. 지능형 시간 관리
- **실시간/고정 모드**: 현재 시간을 실시간으로 추적하거나, 특정 과거/미래 시점을 고정하여 비교할 수 있습니다.
- **멀티 슬롯 지원**: 고정 시간 모드에서 최대 2개의 시간 슬롯을 동시에 표시하여 복잡한 시간 협업을 지원합니다.
- **낮/밤 시각화**: 각 시간대의 직관적인 낮(☀️)/밤(🌙) 아이콘 표시 (낮 기준: 06:00 ~ 18:00).

### 2. 🌍 브라우저 및 글로벌 지원
- **다국어 지원 (i18n)**: 한국어와 영어를 지원하며, 실시간으로 언어를 전환할 수 있습니다.
- **크롬 확장 프로그램**: 브라우저 어디서든 클릭 한 번으로 새 탭에서 세계 시계 앱을 실행할 수 있습니다.
- **고정밀 타임존**: 서울, 뉴욕, 런던 등 전 세계 주요 도시의 시간을 정확하게 제공합니다.

### 3. 강력한 검색 및 커스텀
- **현지화된 도시 검색**: 한국어와 영어 모두 지원하는 검색 엔진을 통해 원하는 도시를 즉시 찾을 수 있습니다.
- **커스텀 오프셋**: 특정 도시가 아닌 수동 UTC 오프셋을 직접 설정하여 시간대를 만들 수 있습니다.
- **그룹 관리**: 목적에 따라 여러 시간대를 그룹으로 묶어 탭으로 관리할 수 있습니다.

### 4. 유틸리티 도구 및 생산성
- **시간 단위 변환기**: 일, 시, 분, 초 간의 상호 변환을 지원합니다.
- **기능 계산기**: 기간 계산(두 날짜 사이의 일수) 및 날짜 이동(N일/주/월 후) 기능을 제공합니다.
- **원클릭 복사**: 개별 행 또는 전체 목록을 정돈된 텍스트 형식(한국어/영어 대응)으로 즉시 복사할 수 있습니다.

## 🚀 시작하기

### 웹 버전 (Web)
별도의 서버 설치 없이 `index.html` 파일을 브라우저에서 여는 것만으로 즉시 사용 가능합니다.

### 크롬 확장 프로그램 (Chrome Extension)
1. 크롬 브라우저에서 `chrome://extensions`에 접속합니다.
2. 우측 상단의 **개발자 모드**를 활성화합니다.
3. **압축해제된 확장 프로그램을 로드** 버튼을 클릭하고 프로젝트 폴더를 선택합니다.
