# 글로벌 타임 뷰어 v3.2.11

글로벌 타임 뷰어는 여러 시간대를 한 화면에서 비교하고, 고정 시간 기준 편집과 기간 계산, 복사/내보내기까지 지원하는 브라우저 기반 도구입니다.

## 주요 기능

### 1) 시간대 비교
- 실시간 모드 / 고정 시간 모드 지원
- 그룹 단위 시간대 관리
- 그룹별 기준 시간대 저장
- 시간대 행 드래그 정렬
- 커스텀 시간대(약어/이름/UTC 오프셋) 추가

### 2) 시간 편집
- 기준 시간대 기준으로 메인/추가 시간 편집
- 빠른 이동 버튼 제공:
  - 지금 시간, 0시, 정각
  - ±1시간, ±1일, ±1주
- 추가 시간(2슬롯)으로 기간 반영 시뮬레이션 가능

### 3) 표기/복사 양식
- 표기/복사 항목 선택 및 순서 드래그 정렬
- 항목: `시간대`, `이름`, `UTC`, `시간`, `일수`, `기간`
- `시간` 항목 세부 체크:
  - `낮/밤`, `날짜`, `시간`, `요일`
- 복사 미리보기 제공

### 4) 계산기 도구
- 시간 단위 변환
- 기간 계산(일/시간/분/초)
- 날짜 이동 계산(일/주/개월)
- 계산 결과 복사

### 5) 내보내기/가져오기
- 설정 JSON 내보내기 / 가져오기
- 시간대 테이블 PNG 저장
- 테마/언어/옵션 상태 로컬 저장

## 실행 방법

### 웹 페이지로 실행
1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.

### Chrome 확장 프로그램으로 실행
1. `chrome://extensions` 접속
2. 우측 상단 `개발자 모드` 활성화
3. `압축해제된 확장 프로그램 로드` 클릭
4. 이 프로젝트 폴더 선택

## 데이터 저장

아래 키로 `localStorage`에 상태를 저장합니다.

- `GTV_v323_Data`
  - 그룹/시간대 목록
  - 그룹별 기준 시간대
  - 표시/복사 양식 및 시간 세부 옵션
  - 실시간/고정 모드 및 각종 토글 상태
- `GTV_Lang`: 언어
- `GTV_Theme`: 테마
