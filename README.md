# Global Time Viewer v3.2.4

여러 시간대를 한 화면에서 비교하고, 고정 시간 시뮬레이션과 기간 계산까지 지원하는 브라우저 기반 도구입니다.
크롬 확장 프로그램으로도 실행할 수 있습니다.

## 주요 기능

### 시간대 비교
- 실시간 시간 / 고정 시간 모드 지원
- 그룹 단위 시간대 관리
- 시간대 행 드래그로 순서 변경
- 기준 시간대 지정 기능
- 기준 시간대는 목록 최상단 고정

### 시간 슬롯
- 기본 시간 + 추가 시간(2슬롯) 표시
- 고정 시간에서 슬롯별 빠른 시간 수정
- 시간 조작 버튼
  - 지금 시간
  - 0시
  - 정각
  - -1시간 / +1시간
  - -1일 / +1일
  - -1주 / +1주

### 복사/표기 제어
- `표기 및 복사 방식` 옵션 지원
- 항목별 표시/복사 포함 여부 제어
  - 시간대
  - 국가 - 지역
  - UTC OFFSET
  - 시간
  - 기간(일수)
  - 기간(시간)
- 항목 드래그로 복사 순서 변경
- 전체 복사 / 행별 복사

### 기간 표시
- 추가 시간이 표시될 때 기간(일수) 계산
- 추가 시간이 표시될 때 기간(일/시간/분) 계산 (예: `1d 3h 20m`, `1일 3시간 20분`)

### 이미지 저장
- 현재 시간대 테이블을 PNG로 저장
- 파일명 규칙:
  - `그룹명_시간대약어_YYYY-MM-DD HHMMSS`
  - 예: `기본 그룹_KST_2026-03-02 165936.png`

### 계산기 도구
- 시간 단위 변환(일/시/분/초)
- 기간 계산(총 일수/시간/분/초)
- 날짜 이동 계산(일/주/월)
- 계산 결과 복사

### UI/환경
- 한국어 / 영어 전환
- 다크 / 라이트 테마
- 마지막 탭/그룹/옵션 상태 저장(localStorage)

## 설치 및 실행

### 웹 페이지로 실행
1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.

### 크롬 확장 프로그램으로 실행
1. 크롬에서 `chrome://extensions` 진입
2. 우측 상단 `개발자 모드` 활성화
3. `압축해제된 확장 프로그램을 로드` 클릭
4. 이 프로젝트 폴더 선택

## 저장 데이터

localStorage에 사용자 상태를 저장합니다.

- `GTV_v323_Data`
  - 그룹 목록
  - 활성 탭/그룹
  - 기준 시간대
  - 추가 시간 표시
  - 시간 수정 표시
  - 표기 및 복사 방식(순서/체크)
- `GTV_Lang`: 언어
- `GTV_Theme`: 테마

이전 버전 키(`GTV_v310_Data` 등)는 자동 마이그레이션됩니다.
