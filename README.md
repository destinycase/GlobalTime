# 🌐 Global Time Viewer v2.0.0

전 세계 여러 도시의 시간을 한눈에 비교하고 관리할 수 있는 프리미엄 세계 시간 뷰어입니다. 세련된 다크 모드 디자인과 강력한 계산 도구를 제공합니다.
## ✨ 주요 기능

### 1. 지능형 시간 관리
- **실시간/고정 모드**: 현재 시간을 실시간으로 추적하거나, 특정 과거/미래 시점을 고정하여 비교할 수 있습니다.
- **멀티 슬롯 지원**: 고정 시간 모드에서 최대 2개의 시간 슬롯을 동시에 표시하여 복잡한 시간 협업을 지원합니다.
- **낮/밤 시각화**: 각 시간대의 직관적인 낮(☀️)/밤(🌙) 아이콘 표시 (낮 기준: 06:00 ~ 18:00).

### 2. 강력한 검색 및 커스텀
- **전 세계 도시 검색**: 광범위한 타임존 데이터베이스를 통해 원하는 도시를 즉시 추가할 수 있습니다.
- **커스텀 오프셋**: 특정 도시가 아닌 수동 UTC 오프셋을 직접 설정하여 시간대를 만들 수 있습니다.
- **그룹 관리**: 목적에 따라 여러 시간대를 그룹으로 묶어 탭으로 관리할 수 있습니다.

### 3. 유틸리티 도구
- **시간 단위 변환기**: 일, 시, 분, 초 간의 상호 변환을 지원합니다.
- **기간 계산기**: 두 날짜 사이의 일수를 계산합니다.
- **날짜 이동 계산**: 특정일로부터 N일/주/월 후의 날짜를 계산합니다.

### 4. 생산성 강화
- **원클릭 복사**: 개별 행 또는 전체 목록을 정돈된 텍스트 형식으로 즉시 복사할 수 있습니다.
- **반응형 디자인**: 매우 높은 정보 밀도를 유지하면서도 좌측 정렬 기반의 깔끔한 가독성을 제공합니다.

## 🚀 시작하기

별도의 서버 설치 없이 `index.html` 파일을 브라우저에서 여는 것만으로 즉시 사용 가능합니다.

```bash
# 로컬에서 실행
open index.html
```

## 🛠️ 개발 및 품질 관리

프로젝트의 유지보수와 코드 품질을 위해 **Node.js** 환경의 최신 도구들을 사용합니다. 자세한 내용은 [README_DEV.md](./README_DEV.md)를 참고하세요.

- **테스트 코어**: [Vitest](https://vitest.dev/)를 통한 로직 검증
- **린트/문법**: [ESLint](https://eslint.org/)를 이용한 코드 스타일 강제 및 버그 예방
