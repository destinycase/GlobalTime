# Global Time Viewer v3.3.1

Global Time Viewer는 여러 시간대를 빠르게 비교하고, 시간 조정/기간 계산/복사/이미지 저장까지 처리할 수 있는 브라우저 기반 도구입니다.

## 주요 기능

### 1) 시간대 비교
- 실시간 모드 / 시간 변경 모드 지원
- 그룹 단위 시간대 관리
- 기준 시간대 선택
- 시간대 행 드래그 정렬
- 커스텀 시간대(약어/이름/UTC 오프셋) 추가

### 2) 구간 시간 변경
- 탭 명칭: `구간 시간 변경`
- 구간 수량: 1~100개 (`+ / -` 버튼 및 숫자 입력)
- 구간 이름 사용자 지정
- 각 구간의 시작/끝 시간 연쇄 계산
- 구간 시작/구간 일괄 조정 도구
  - 지금 시간, 0시, 정각
  - -1시간, +1시간, -1일, +1일, -1주, +1주, -4주, +4주
  - 커스텀 일수
  - `기간 0일로 변경`
- 구간 단위 복사, 이하 구간 접기/펼치기, 이 구간 접기/펼치기

### 3) 표기/복사 양식
- 표기/복사 항목 선택 및 순서 드래그 정렬
- 항목: 시간대, 이름, UTC, 시간, 일수, 기간
- 시간 하위 항목: 주야, 날짜, 시간, 요일
- 복사 미리보기 제공

### 4) 계산기 도구
- 시간 단위 변환
- 기간 계산(일/시간/분/초)
- 날짜 이동 계산(일/주/월)
- 결과 복사

### 5) 이미지 저장
- 기본 테이블 이미지 저장
- 구간 시간 변경 전용 저장 모드
  - `이미지 저장 - 구간만`: 구간 타이틀만 1장 저장
  - `이미지 저장 - 구간별`: 각 구간을 개별 파일로 저장
- 이미지 렌더링 시 숫자 고정폭 폰트 적용

### 6) UI/설정
- UI 스케일: `50% / 75% / 100% / 125% / 150% / 175% / 200%`
- 테마: 다크 / 라이트
- 언어: 한국어 / English
- 읽기 전용 시간 입력(실시간 시간, 2구간 이상 시작 시간) 시각 구분 강화

## 실행 방법

### 웹페이지로 실행
1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.

### Chrome 확장 프로그램으로 실행
1. `chrome://extensions` 접속
2. 우측 상단 `개발자 모드` 활성화
3. `압축해제된 확장 프로그램 로드` 클릭
4. 프로젝트 폴더 선택

## 데이터 저장
다음 키로 `localStorage`에 상태를 저장합니다.
- `GTV_v323_Data`: 그룹/시간대/옵션/구간 상태
- `GTV_Lang`: 언어
- `GTV_Theme`: 테마
- `GTV_UIScale`: UI 스케일
